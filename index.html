LVG Bioinfo resources list

A list of bioinformatic resources and tools to get started with bioinfo at the LVG

Unix

  • Unix cheatsheet | The famous PDF with basics commands for bioinfo
  • UNIX complete tutorial | Tutorial One, Two and Three are the basics
  • SIB tutorial | Interactive tutorial about UNIX basics

Bedtools

For installing BEDtools, please do ask us to help you. Installation can be a bit tricky and you can mess up your computer.

  • BEDTools suite | List of all the bedtools tools with explanation
  • BEDTools tutorial | Examples of main usages

NGS and Omics analysis

  • Wikibooks on NGS | Very complete wikibook about NGS data - start here
  • Video: High-Throughput Sequencing | Small introduction video about sequencing

RNA-seq

  • RNA-seqlopedia | All you need to know about RNA-seq

ChIP-seq

  • ChIP-seq talk | Talk by Remco Loos from the EMBL-EBI (slides+vid); old but useful
  • Complete MIT OpenCourse | Detailed course on ChIP-seq; full of math but start from the beginning

Programming language

At some point, you have to dive in... An advice I would give you is: try to have a small project you would like to do which requires a bit of coding. The learning curve is a sigmoid function which means you will struggle a LOT at the beginning. Having a project keeps you motivated and there is nothing more rewarding than to finally solve all the problems you had an complete your project.

  • Codeacademy - Learn to code interactively | Learn different languages interactively

    • Python

    • Python Beginner Guide | Official python's wiki
    • Python interactive tuto | Try python and learn at the same time; easiest to start
    • Learn python the hard way | Probably one of the best method, but also the hardest... start to finish
    • Datacamp - lilke military camp but better | Interactive course in the browser for data analysis

    R

    • R Bioconductor manual | Very good R manual from bioconductor; good but hard to read
    • Official manual | Standard R manual
    • Interactice tutorial | Learn R basics easily and interactively
    • Swirl - Learn R, in R | Package to learn R basics directly in R
    • Datacamp - lilke military camp but better | Interactive course in the browser for data analysis

    Books

    • Biomedical Data Science | This book will cover several of the statistical concepts and data analytic skills needed to succeed in data-driven life science research

    Other but very useful

    • Google | Everything has already been done; you'll discover stack overflow, biostar and so on...
    • Rosalind | Platform to learn and train bioinfo; full of small exercises to learn
